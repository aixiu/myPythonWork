- JSON 指的是 JavaScript 对象表示法（JavaScript Object Notation）
- JSON 是轻量级的文本数据交换格式
- JSON 独立于语言 *
- JSON 具有自我描述性，更易理解

- JSON 语法规则
- JSON 语法是 JavaScript 对象表示法语法的子集。

  - 数据在名称/值对中
  - 数据由逗号分隔
  - 花括号保存对象
  - 方括号保存数组

  ```json
    {
    "employees": [
    {"firstName":"John" , "lastName":"Doe"},
    {"firstName":"Anna" , "lastName":"Smith"},
    {"firstName":"Peter" , "lastName":"Jones"}
    ]}
    ```
